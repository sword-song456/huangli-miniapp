# 节气老黄历小程序 - 部署指南

## 一、准备工作

### 1. 注册小程序账号
- 访问 [微信公众平台](https://mp.weixin.qq.com/)
- 注册小程序账号
- 获取 AppID

### 2. 安装开发工具
- 下载 [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)
- 安装并登录

### 3. 申请API密钥

#### 黄历数据API（二选一）
**聚合数据（推荐）**
- 网址：https://www.juhe.cn/
- 注册账号
- 申请"万年历"API
- 获取 API Key

**天行数据**
- 网址：https://www.tianapi.com/
- 注册账号
- 申请"老黄历"API
- 获取 API Key

#### 天气API（可选）
**心知天气**
- 网址：https://www.seniverse.com/
- 注册账号
- 获取免费API Key

## 二、配置项目

### 1. 修改 project.config.json
```json
{
  "appid": "你的小程序AppID"
}
```

### 2. 配置API密钥
修改 `utils/api.js` 文件：
```javascript
const API_CONFIG = {
  juheKey: '你的聚合数据API Key',
  tianxingKey: '你的天行数据API Key',
  weatherKey: '你的天气API Key'
}
```

### 3. 配置广告位
修改以下文件中的广告单元ID：
- `pages/index/index.wxml`
- `pages/card/card.wxml`

将 `unit-id="adunit-xxxxxx"` 替换为你的广告位ID

## 三、开发调试

### 1. 导入项目
- 打开微信开发者工具
- 选择"导入项目"
- 选择 `huangli-miniapp` 文件夹
- 填入 AppID

### 2. 配置服务器域名
在小程序后台配置以下域名：
- request合法域名：
  - `https://v.juhe.cn`（聚合数据）
  - `https://api.tianapi.com`（天行数据）
  - `https://api.seniverse.com`（心知天气）

### 3. 本地调试
- 点击"编译"按钮
- 在模拟器中测试功能
- 检查控制台是否有错误

## 五、上线发布

### 1. 代码审核
- 点击"上传"按钮
- 填写版本号和项目备注
- 提交审核

### 2. 审核注意事项
- 确保所有功能正常
- 广告位不能遮挡主要内容
- 不能有诱导分享行为
- 隐私政策要完善

### 3. 发布上线
- 审核通过后
- 在小程序后台点击"发布"
- 设置体验版供测试

## 六、运营优化

### 1. 流量主开通
- 累计独立访问用户达到1000
- 在小程序后台申请开通流量主
- 配置广告位

### 2. 数据分析
- 关注日活跃用户数
- 分析分享转化率
- 优化用户留存

### 3. 功能迭代
- 添加更多节气知识
- 优化平安卡样式
- 增加个性化设置

## 七、常见问题

### API调用失败
- 检查API Key是否正确
- 确认服务器域名已配置
- 查看API调用次数是否超限

### 图片生成失败
- 检查Canvas尺寸设置
- 确认授权相册权限
- 调试绘图代码

### 广告不显示
- 确认流量主已开通
- 检查广告位ID是否正确
- 查看广告填充率

## 八、成本预估

### 免费额度
- 聚合数据：100次/天（免费版）

- 日活1000人：约需500次API调用
